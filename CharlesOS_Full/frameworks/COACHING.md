# Coaching Framework

## Core Principle

Kids learn best when they are active, engaged and enjoying the session.

## Coaching Rules

- Keep interventions to 30–45 seconds
- Avoid long queues
- Avoid players standing around
- Use simple language
- Praise effort
- Encourage bravery
- Correct one thing at a time
- Let them play
- Use repetition with small variations

## Session Design

Every session should include:

1. Movement
2. Ball touches
3. Decision-making
4. Competition
5. Game realism
6. Fun

## Intervention Format

Use:

1. Freeze briefly
2. Ask one question
3. Show one example
4. Restart quickly

## Coaching Tone

- Positive
- Energetic
- Clear
- Encouraging
- Consistent
