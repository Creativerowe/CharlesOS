# AFC Shortlands

## Context

Charles coaches AFC Shortlands U12 girls football.

## Coaching Philosophy

- Keep players active
- Avoid standing around
- Use short 30–45 second interventions
- Make learning fun
- Use encouragement, high-fives, fist bumps and positive energy
- Keep instructions simple
- Build confidence through repetition
- Prioritise consistency over complexity

## Core Themes

- Passing
- Control
- Triangle support
- Making space
- Tackling
- Pressing
- Covering
- Communication
- First touch
- Head up

## Useful Phrases

- Pass. Control. Think.
- Move to receive.
- First touch, head up.
- If you lose it, go get it back.
- When in doubt, smash it out.

## Team Identity

- Defence: The Wall
- Midfield: The Engine
- Attack: The Strike

## Session Structure

Training should usually include:

1. Fast arrival activity
2. Technical warm-up
3. Small-sided practice
4. Game-realistic challenge
5. Match play
6. Short recap

## Rules for Session Design

- Minimal queues
- Lots of touches
- Small groups
- Clear success criteria
- Simple progressions
- No long lectures
- Let the game teach where possible
